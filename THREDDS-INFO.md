# NOAA THREDDS Data Server - Real-Time Level 2 Access

## What is THREDDS?
THREDDS (Thematic Real-time Environmental Distributed Data Services) is NOAA's official data distribution system. It provides:
- ✅ Real-time Level 2 NEXRAD data
- ✅ Direct HTTPS download (no cloud provider needed)
- ✅ No authentication required
- ✅ Updated every 4-6 minutes
- ✅ Used by professional weather services

## THREDDS URLs for NEXRAD Level 2

Base URL: https://thredds.ucar.edu/thredds/

Catalog for each station:
- KLWX: https://thredds.ucar.edu/thredds/catalog/nexrad/level2/KLWX/catalog.html
- KDIX: https://thredds.ucar.edu/thredds/catalog/nexrad/level2/KDIX/catalog.html

Direct file download format:
https://thredds.ucar.edu/thredds/fileServer/nexrad/level2/SITE/YYYYMMDD/Level2_SITE_YYYYMMDD_HHMMSS.ar2v
